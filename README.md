# docker_machine_provisioning

## How to install
```
curl -sL bit.ly/dockerM > ./mgmt.sh && chmod +x ./mgmt.sh;
sudo mv ./mgmt.sh /usr/local/bin/

```

## How to use
```
==============================================================
 What do you want ?
==============================================================
[0] Install the Virtualbox & docker-machine
[1] Create VM & login
[2] Login to VM
[3] Stop VM
[4] Start VM
[5] Reboot VM
[RM] Remove VM
==============================================================
 Please insert a key as you need = 0
```
